# Pasasoft

## Create a library

lerna create components

## Add a dependencies

lerna add typescript --dev

lerna add @types/slate --scope=@pasa-design/editor --dev

## Install dependencies

lern bootstarp

## Genarate *.d.ts files

lerna run tsc

## Publish npm library

lerna run build

lerna changed

lerna version

lerna publish

## Origanization npm library

npm publish --access public
